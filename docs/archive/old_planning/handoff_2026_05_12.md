## 6. 진행 우선순위 (다음 1–2 주)

### ★★★ 1순위: FNO 학습 (RunPod A100)

코드 완전 준비됨. 실행만 하면 됨.

```bash
# RunPod A100 환경에서
python -m src.training.train_fno --dataset data/processed/dataset.h5 \
    --output checkpoints/fno_no_pi --epochs 100 --batch-size 8 \
    --wandb --wandb-name fno_no_pi_a100

python -m src.training.train_fno --dataset data/processed/dataset.h5 \
    --use-pi --resume checkpoints/fno_no_pi/best.pt \
    --output checkpoints/fno_pi --epochs 100 --batch-size 8 \
    --wandb --wandb-name fno_pi_a100
```
